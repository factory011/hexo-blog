边框圆角

使用css画一个圆：border-radius（边框圆角）

border：边框

radius：半径

取值：左上、右上、右下、左下（顺时针赋值）

  • 画一个圆（盒子必须是正方形）

  • 胶囊按钮（矩形，border-radius为高度的一半）
